## ğŸ¯ Modal Navigation

All modals support the following:

| Shortcut | Action |
|----------|--------|
| `Tab` | Focus next field |
| `Shift+Tab` | Focus previous field |
| `Enter` | Submit form (when on button) |
| `Esc` | Close modal |

**Focus Trap:** When a modal is open, Tab navigation stays within the modal and doesn't escape to the background.

## ğŸ’¡ Tips

2. **Esc key always closes**: Whenever you're in a modal or search, `Esc` will close it.

4. **Mobile users**: All features are accessible via touch, no shortcuts needed!
